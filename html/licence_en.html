Anonymity

To maintain anonymity at this server, neither PONK, nor the external services UDPipe and NameTag, log the processed texts.
The only information that may be logged is: time of usage, size of the processed data, the system configuration and the IP address from where it is accessed; if you use the service via REST API, make sure to use the POST method, as REST API URLs may also be logged.
