PONK je webový nástroj a služba REST API pro analýzu srozumitelnosti českých právních textů. Vstupní text (např. dopis zaslaný správním orgánem občanovi) je zpracován pomocí externích služeb pro větný rozbor a rozpoznání jmenných entit. Program poté identifikuje a zvýrazní části textu s nízkou srozumitelností.
